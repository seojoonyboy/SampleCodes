GunfightKing Mobile FPS Game
==========================
> 모비릭스 모바일 FPS 클라이언트 개발   
> 개발 기간 : 2024.07 ~ 2025.11   
> 출시 여부 : Google PlayStore, Apple Store 정식 런칭

개발 환경
==========================
엔진 : Unity 3d Engine 6000.0.59f1   
플랫폼 : Android, iOS   
버전 관리 : Git, Github

프로젝트 소개
==========================
*총싸움의 왕 특징*
> 실시간 PVP 방식의 모바일 FPS 게임
> PVP 방에서 AI (Bot)이 섞여서 함께 플레이를 하기도 한다.
