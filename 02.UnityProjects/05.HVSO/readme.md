개발 환경
==========================
엔진 : Unity 2018.4.12f1      
플랫폼 : Android, iOS   
버전 관리 : Git, Gitlab      


프로젝트 소개
==========================
*HVSO 특징*   
실시간 턴제 기반 전략 카드 게임.   
유저와 유저를 서로 매칭하여 실시간으로 PVP를 즐길 수 있고, 
AI와 대전하는 PVE 컨텐츠가 있음.    
게임 방식은 기존 EA사의 플랜츠 vs 좀비와 유사하다.

===========================
